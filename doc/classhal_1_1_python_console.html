void hal::PythonConsole::keyPressEvent ( QKeyEvent *  e)
overrideprotectedvirtual

Overrides QTextEdit::keyPressEvent. It is used to handle the following (default) inputs:

  • Ctrl+C: Copy
  • Ctrl+X: Cut
  • ENTER/RETURN: Interpret the written python command
  • BACKSPACE: Backspace
  • KEY_UP: Get the previous command from the history
  • KEY_DOWN: Get the next command from the history
  • TAB: Indent or autofill
