Detailed Description

The interface for the logical part of a setting.

The SettingsItem class provides an interface and commonly shared function for all kinds of specific settings. It is the logical part of a setting in comparison to the SettingsWidget, which represents the corresponding visual part. It -more precise the specific settings- connects to the SettingsManager.
