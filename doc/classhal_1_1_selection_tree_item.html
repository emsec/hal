Detailed Description

Basic item for the SelectionTreeModel.

The basic tree item class for the SelectionTreeModel. It is the base class for every other subitem (except for SelectionTreeItemRoot) as it provides all commonly shared and necessary functions for the model and related events.

◆ suppressedByFilterRecursion()

virtual void hal::SelectionTreeItem::suppressedByFilterRecursion ( QList< u32 > &  modIds,
QList< u32 > &  gatIds,
QList< u32 > &  netIds,
const QRegularExpression &  regex 
) const
pure virtual

A function that fills the modIds, gateIds, and netIds lists with the respective ids if the called selection-tree-item subclass does not match the given regex (and in thus suppressed, meaning should not be displayed). Must be implemented by the specific subclass since a module-type item must call this function of its children. A gate- or net-type item simply adds itself to the list (or not).

Parameters
modIds- The list that contains the suppresed module-type items at the end.
gatIds- The list that contains the suppressed gate-type items at the end.
netIds- The list that contains the suppressed net-type items at the end.
regex- The regular expression to match the items against.

Implemented in hal::SelectionTreeItemNet, hal::SelectionTreeItemGate, and hal::SelectionTreeItemModule.
