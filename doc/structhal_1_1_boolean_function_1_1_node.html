Detailed Description

Node refers to an abstract syntax tree node of a Boolean function. A node is an abstract base class for either an operation (e.g., AND, XOR) or an operand (e.g., a signal name variable).

u16 hal::BooleanFunction::Node::get_arity ( u16  type)
static

Returns arity of node, i.e how many operand nodes does the node have. For example, an 'And' node has an arity of 2 as it has 2 operands, a 'Not' node has an arity of 1 respectively. Operand nodes such as a 'Constant' or 'Variable' have an arity of 0.
