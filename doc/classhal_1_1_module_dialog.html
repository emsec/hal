Detailed Description

The ModuleDialog class opens a popup window for module selection.

This feature is mainly used to select the target for the 'move to module...' action. Dialog returns 'accepted' upon any selection or when pushing 'Create new module' Dialog returns 'rejected' upon Cancel button and 'Pick from graph' button
