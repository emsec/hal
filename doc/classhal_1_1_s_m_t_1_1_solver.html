Detailed Description

SMT::Solver is used to query SMT solvers for a list of constraints, i.e. statements that have to be equal. To this end, we translate constraints to a SMT-LIB v2 string representation and query solvers with a defined configuration, i.e. chosen solver, model generation etc.
